1. tkinter, pandas 등 필요한 모듈을 설치한다.
2. secrets.py 파일에 API 키 (consumer_key, consumer_secret, access_token, access_token_secret) 4개를 입력한다
3. tweets_analyzer.py 파일의 터미널에서 "python tweets_analyzer.py -n urimal365(트위터 주소, @ 생략) -l(트윗 개수 조절) 20(20을 넘어서 1000 이상도 가능)"을 입력한다
4. gui에서 버튼으로 각종 정보를 확인할 수 있으며 '엑셀출력' 버튼 클릭 시 tweets_analyzer.py파일이 있는 폴더에 해당 정보가 담긴 엑셀 파일이 저장된다.
