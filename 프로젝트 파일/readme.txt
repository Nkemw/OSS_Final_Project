2. 압축 해제하여 나온 폴더 안에 tweets_analyzer2.zip, tweets_znalyzer3.zip을 압축 해제한다
3. tweets_analyzer.exe 파일을 실행한다.
4. 트윗 수는 기본 1000으로 고정되어 있고 트윗 수 버튼을 통해 트윗 수를 조절한다.
5. 트윗 수 버튼 우측의 텍스트 공간에 트위터 주소를 입력한다(@ 생략)
6. gui에서 버튼으로 각종 정보를 확인할 수 있으며 '엑셀출력' 버튼 클릭 시 tweets_analyzer.py파일이 있는 폴더에 해당 정보가 담긴 엑셀 파일이 저장된다.
